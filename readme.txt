Для запуска программы из командной строки Windows:
1. Установить Java SE Development Kit
2. Добавить в значение переменной среды PATH путь к файлам javac.exe и java.exe
3. Скопировать проект в любую папку
4. В папку src\ добавить файлы необходимые для сортировки слиянием
5. В командной строке Windows переместится в папку src, "cd <диск>\<путь к проекту>"
6. Скомпилировать проект, ввести в командной строке: "javaс ru\cft\khanov\Main.java"
7.1. В случае появления сообщения: "не является внутренней или внешней командой..."
перейти на шаг 2 и проверить правильность пути к файлам javac.exe и java.exe
7.2. В случае появления сообщения: "Note: Some input files use unchecked or unsafe operetions..." - перейти на шаг 6
8. Запустить проект, ввести в командной строке:
 "java ru.cft.khanov.Main <ключ параметр сортировки> <ключ тип данных> <имя результирующего файла> <входной файл 1>...<входной файл N>"

Параметры программы задаются при запуске через аргументы командной строки, по порядку:
1. режим сортировки (-a или -d), необязательный, по умолчанию сортируем по возрастанию;
2. тип данных (-s или -i), обязательный;
3. имя выходного файла, обязательное;
4. остальные параметры – имена входных файлов, не менее одного.
Примеры запуска из командной строки для Windows:
java ru.cft.khanov.Main -i -a out.txt in.txt (для целых чисел по возрастанию)
java ru.cft.khanov.Main -s out.txt in1.txt in2.txt in3.txt (для строк по возрастанию)
java ru.cft.khanov.Main -d -s out.txt in1.txt in2.txt (для строк по убыванию)
